# Form Builder

Tired of complex survey tools like Google Forms? Here's my app for you!

![alt text](image.png)

![form constructor](image-1.png)

## Tech stack

```
VUEJS 3(COMPOSITION API) + VUEX + TAILWINDCSS + DAISYUI + VITE + ESLINT + PRETTIER
```
